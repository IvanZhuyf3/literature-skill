# The high resolution spectrum of $\mathrm { A s H } _ { 3 }$ (4 0 0) local mode state: symmetry reduction and rotational re-quantization

Ji-Xin Cheng \*, Xiao-Gang Wang, Hai Lin, Qing-Shi Zhu

Department of Chemical Physics, Uni6ersity of Science and Technology of China. Hefei, Anhui, , People s Republic of China

Received 12 April 1998; received in revised form 21 April 1998; accepted 21 April 1998

## Abstract

The (4 0 0) stretching overtone band of $\mathrm { A s H } _ { 3 }$ in the range of 7900– $8 1 0 0 ~ \mathrm { c m } ^ { - 1 }$ was recorded by a Fourier transform spectrometer with a resolution of $0 . 0 1 ~ \mathrm { c m } ^ { - 1 }$ . The $\mathbf { A } _ { 1 }$ or $\mathbf { A } _ { 2 } - \mathbf { E }$ degeneracy of the upper state rotational levels up to J% = 8 was observed. It is interesting to find that this spectrum resembles an $A ,$ C hybrid band of an asymmetric top and can be assigned by only one set of quantum numbers $J , K _ { \mathrm { a } }$ and $K _ { \mathrm { c } } .$ The rotational levels were determined by combination difference method and fitted in both the symmetric top and the asymmetric top models. In the latter case, the effective rotational constants A, B and C are in good agreement with the values predicted by a simple local mode model, indicating that the dynamical symmetry of $\mathrm { A s H } _ { 3 }$ in the (4 0 0) local mode state has reduced to that of an asymmetric top. The transition from normal mode to local mode vibration from 6=1 to 4 in AsH was discussed. © 1998 Elsevier Science B.V. All rights reserved.

Keywords AsH ; Symmetry reduction; Rotational re-quantization

## 1. Introduction

The rotational energy level structure of the stretching overtone states of small symmetric molecules is of great interest due to its unusual features near the local mode limit. First, it has been demonstrated that there exist simple relations (so called ‘a-relations’) between the effective vibration–rotation parameters in the normal mode approach [1–3]. The other peculiarity is the reduction of dynamical symmetry [2,4–7], that is, $\mathrm { T } _ { d } {  } C _ { 3 v } C _ { 3 v } {  } C _ { s }$ and $C _ { 2 v }  C _ { \mathrm { s } }$ for $X Y _ { n } , ( n =$

4, 3, 2) molecules respectively. In detail, the spectrum of a $\mathrm { T } _ { d }$ molecule in the local mode state should resemble a parallel band of a symmetric top molecule and can be assigned by a set of rotational quantum numbers (J, K) of symmetric tops. This interesting phenomenon has been observed in GeH (n 0 0 0), n=3 [8] and 5 [9], SiH (n 0 0 0), n=3, 4 and 5 [10,11] and SnH (n 0 0 0), n = 7 [12] and 6, 8 [13] states.

For $C _ { 3 v }$ molecules, the rotational analysis on the 6=2 [14], 3 [15] and 4 [16] stretching overtones of SbH provided good examples of the local mode features in $X Y _ { 3 }$ molecules, i.e. the disappearance of the Coriolis couplings and the simple arithmetic relations between some vibration–rotation parameters. The recent FT study [17] of $\mathrm { G e H } _ { 3 } \mathrm { D } \ ( v = 3 )$ and ICLAS study [18] of $\mathrm { G e H } _ { 3 } \mathrm { D } \ \left( v = 6 , 7 , 8 \right)$ together with $\mathrm { S b H } _ { 3 } ~ ( v = 4 )$ [16] showed the rotational energy level pattern is close to that of an asymmetric rotor.

AsH is a typical pyramid molecule. One expects to be able to observe the local mode behavior in its stretching overtone states. The overtone spectra with medium resolution have been recorded and analyzed with a local mode model [19]. Later Fermi resonance between the stretching modes and the bending modes was investigated [20]. The first [21] and second [22] stretching overtone bands have been rotationally analyzed. In the (3 0 0) state, the $\mathbf { A } _ { 1 ( 2 ) } - \mathbf { E } \ ( \mathbf { A } _ { 1 ( 2 ) }$ denotes $\mathbf { A } _ { 1 }$ or ${ \bf A } _ { 2 } )$ 3-fold degeneracy of the rotational levels has been observed and an analysis in the asymmetric top model demonstrated the tendency of symmetry reduction from $C _ { 3 v }$ to $C _ { s }$ [22]. In the present contribution we present a detailed rotational analysis of its third overtone state. It is found that the AsH (4 0 0) state is a good example of symmetry reduction and rotational re-quantization of symmetric top molecules at the local mode limit.

## 2. Experimental

$\mathrm { A s H } _ { 3 }$ with purity 99.99% was purchased from Nanjing Special Gas Company. A high resolution spectrum was recorded at room temperature on the Bruker 120 HR interferometer, which was equipped with a $\mathrm { C a F } _ { 2 }$ beam splitter, a InSb detector and a tungsten halogen source. A sample pressure of 33 torr was chosen. No optical filter was applied. A multi-pass absorption cell with effective path length of 105 m was used. Scans (2250) were co-added. The apodized resolution $0 . 0 1 ~ \mathrm { ~ c m } ^ { - 1 }$ (MOPD) is comparable with the Doppler width $0 . 0 1 1 ~ \mathrm { c m } ^ { - 1 }$ . The precision of the line position was estimated to be better than 0.001 $\mathrm { c m } ^ { - 1 }$

The spectral range from 5900 to 8900 $\mathrm { c m } ^ { - 1 }$ was covered. The spectrum from 7900 to 8100 $\mathrm { c m } ^ { - 1 }$ was assigned to the (4 0 0) stretching overtone band according to the local mode model calculation (see below). The other two weaker bands in the range of 6120– $6 5 0 0 ~ \mathrm { \ c m } ^ { - 1 }$ and 8130–8350 $\mathrm { c m } ^ { - 1 }$ were assigned to $( 2 1 0 )$ and (3 1 0) transitions respectively. The calibration was carried out by measuring the residual $_ { \mathrm { H } _ { 2 } \mathrm { O } }$ lines in the range of $7 3 0 0 { - } 7 6 0 0 ~ \mathrm { c m } ^ { - 1 }$ [23]. The accuracy of the measurements is better than 0.001 $\mathrm { c m } ^ { - 1 } [ 2 4 ]$ .

## Description and assignment of the spectrum

The (4 0 0) band in the range of 7900–8100 $\mathrm { c m } ^ { - 1 }$ as shown in Fig. 1 corresponds to the transitions to the degenerate local mode pair, i.e. $| 4 0 0 ; \mathbf { A } _ { 1 } \rangle$ and 400; E states. The rotational constants of these two states are nearly equal. On the other hand, as a quasi-spherical molecule, the difference between the constants B and C is rather small so that the rotational levels with the same J% are close in energy. As a result the rovibrational levels are strongly coupled by the $H _ { 2 2 }$ terms which are essential for the rotational pattern of local mode state [1–4,7]. In the spectrum, more transitions become visible due to the mixing of the vibration–rotation wavefunctions. Fig. 2 shows an example of the doubling of some transitions in the R(1) and R(2) branches.

The ground state constants were determined by the far infrared spectrum [34]. A refinement has been carried out recently [35]. The ground state (GS) constants used in the present work are taken from Ref. [34], which are found to be adequately accurate for our study. The calculated effective parameters of the upper states were used to predict the rotational energy levels. Besides, the band origins were estimated from the recorded spectrum. Moreover, the quartic centrifugal distortion constants $D _ { J } , D _ { K }$ and $D _ { J K }$ were initially constrained to the ground state values. All the other high-order parameters used in the calculation were set to zero initially.

The selection rules applied for both $\mathbf { A } _ { 1 }$ and E states were $\Delta J = 0 , ~ \pm 1 , ~ \Delta K = 0 , ~ \pm 1 , ~ \pm 2 , ~ \pm 3 .$ , ElE and $\mathbf { A } _ { 1 }  \mathbf { A } _ { 2 } ,$ which produce a combination of parallel and perpendicular transitions. The ground state combination differences (GSCD) linking levels to a common upper rotational state were used for the assignment of the low J% rotational levels. Two interesting features were found and used to verify the assignment. First, as indicated by the prediction, the rotational levels of symmetry $\mathbf { A } _ { 1 ( 2 ) }$ and E with the same J% and K values are degenerate into one (for $K = 0 )$ or two (for $K \neq 0 )$ pairs with splitting $< 0 . 0 1 \ \mathrm { c m } ^ { - 1 }$ . Second, due to the strong mixing of the vibration– rotation wavefunctions, four to five transitions can be found for many rotational levels of the upper states. The assigned rotational levels were then used in the fit to obtain improved values of the spectroscopic parameters and to predict the levels for higher J%. The same process was repeated until $J ^ { \prime } = 8$ . Altogether 317 transitions were assigned.

It should be noted that some high J% levels are accidentally perturbed and the $\mathbf { A } _ { 1 ( 2 ) } - \mathbf { E }$ degeneracy is consequently destroyed. The perturbed levels are marked by \* in Table 3 and not used in the fit. For $J ^ { \prime } \geq 9$ levels, some lines were observed and assigned but not used in the fit, because most assigned rotational levels were found to deviate from the prediction, moreover, the $\mathbf { A } _ { 1 ( 2 ) }$ –E degeneracy can hardly be satisfied. This indicates the existence of perturbation. A preliminary analysis indicates the probable perturber is the $( 3 0 0 ; v _ { 2 } =$ $v _ { 4 } = 1 )$ state. Its band center was estimated to be $\sim 8 0 4 0 . 4 2 \ \mathrm { c m } ^ { - 1 }$ by an addition of the band centers of the (3 0 0) state [22] and the $v _ { 2 } = v _ { 4 } = 1$ state [29].

## 4. Symmetry reduction and rotational re-quantization

Assuming symmetry reduction in the local mode limit, Lehmann established the Hamiltonians in the local mode basis for $X Y _ { n } ~ ( n = 2 , 3 , 4 )$ molecules. Then he obtained the vibration-rotation coupling terms by transforming these Hamiltonians to a symmetrized basis [4]. Alternatively, Child and Zhu derived the effective rotational Hamiltonians of the local mode states by averaging the vibration–rotation Hamiltonians over the corresponding local mode vibrational wavefunctions [6,7]. For an $X Y _ { 3 }$ molecule, the effective rotational Hamiltonian at the local mode limit can be transferred into that of an asymmetric top with three different effective rotational constants [39,7]

$$
\mathrm{A} = \frac {1}{2} \left\{\left(B _ {v} + C _ {v} + E\right) + \left[ \left(B _ {v} - C _ {v} + E\right) ^ {2} + 4 G ^ {2} \right] ^ {\frac {1}{2}} \right\} \tag {9}
$$

$$
B = B _ {v} - E \tag {10}
$$

$$
C = \frac {1}{2} \left\{\left(B _ {v} + C _ {v} + E\right) - \left[ \left(B _ {v} - C _ {v} + E\right) ^ {2} + 4 G ^ {2} \right] ^ {\frac {1}{2}} \right\} \tag {11}
$$

where $\mathbf { B } _ { v }$ and $\mathbf { C } _ { v }$ are the effective rotational constants in the symmetric top Hamiltonian. G and E are related to the a coefficients by the following relations:

$$
E = \left(\frac {\sqrt {2}}{3} \alpha_ {1 3} ^ {x x} - \frac {1}{3} q _ {3}\right) (\langle n | \bar {r} ^ {2} | n \rangle - \langle 0 | \bar {r} ^ {2} | 0 \rangle) \tag {12}
$$

$$
G = \left(\frac {4}{3} r _ {3} + \frac {\sqrt {2}}{3} \alpha_ {1 3} ^ {z x}\right) (\langle n | \bar {r} ^ {2} | n \rangle - \langle 0 | \bar {r} ^ {2} | 0 \rangle) \tag {13}
$$

On the other hand, the effective off-diagonal $H _ { 2 2 }$ parameters $q _ { \mathrm { e f f } } , \ r _ { \mathrm { e f f } } , \ \alpha _ { \mathrm { e f f } } ^ { z x } \mathrm { a n d } \ \alpha _ { \mathrm { e f f } } ^ { x x }$ can be directly related to their corresponding fundamental coefficients [32,3]. Replacing the symmetrized local mode wavefunctions by linear combination of single bond local-mode wavefunctions and transforming the normal coordinates into the bond displacement coordinates, the integration over the vibrational coordinates yields the following relations [3]:

$$
E = - q _ {\text { eff }} = \frac {\sqrt {2}}{2} \alpha_ {\text { eff }} ^ {x x} \tag {14}
$$

$$
G = 4 r _ {\mathrm{eff}} = \frac {1}{\sqrt {2}} \alpha_ {\mathrm{eff}} ^ {z x}. \tag {15}
$$

lt can be easily proved that Lehmann’s result can be transformed into the same forms as Eqs. (9)–(11) under the following relations

$$
q = 2 E, \quad r = 2 G \tag {16}
$$

where q and r are the rotational coupling coefficients in Lehmann’s model [4].

According to the prediction of Ovchinnikova [39] and Lehmann [4], the overtone spectrum should resemble an A, C hybrid band of an asymmetric top at the limit of small inter-bond couplings. The new coefficients with plus subscript are diagonal in the local mode basis, while those with minus subscript describe the rotational coupling between different local mode states. We transformed the observed parameters of $\mathrm { A s H } _ { 3 }$ (2 0 0), (3 0 0) and (4 0 0) bands into the above forms. The results are shown in Table 6.

lt can be seen that those parameters with minus subscript tend to become zero with the increase of the vibrational quantum number from $v = 2$ to 4, however, $D _ { J } , ~ D _ { J K }$ and $D _ { K }$ are exceptions. The probable reason is that, in both (3 0 0) and (4 0 0) state, the $r _ { \mathrm { e f f } }$ parameter was fixed to the theoretical calculation, whose error should be compensated by adjusting the D values in the fit. The above results demonstrate the disappearance of rotational couplings between different local mode states (n 0 0), (0 n 0) and (0 0 n) with the increase of vibrational energy, therefore the rotational level structure should resemble that of an asymmetric rotor. This is equivalent to the analysis in Section 4.

## 6. Discussion

When the vibrational energy is localized in one single bond, the principal axes of the effective rotational moment inertia are rotated by an angle u in the plane defined by the $C _ { 3 }$ axis and the excited bond [4]. As a result, the $C _ { 3 }$ axis is no longer the symmetric axis in the local mode vibration. The projection of the angular momentum along the $C _ { 3 }$ axis cannot be regarded as a commutator of the molecular Hamiltonian any more. The rotation should be re-quantized in this case because the asymmetric top quantum numbers $K _ { \mathrm { a } } , K _ { \mathrm { c } }$ offer a clearer picture to the rotational structure. Using the local mode relation $r = ( A - C )$ sinu derived by Lehmann [4] and Eq. (16), the rotated angle of the effective principal axes is estimated to be $1 6 . 5 ^ { \circ }$ for the (4 0 0) stretching vibration of $\mathrm { A s H } _ { 3 } ,$ which can be compared with $\theta = 1 3 . 9 ^ { \circ }$ in the (3 0 0) state of $\mathrm { A s H } _ { 3 }$ [22].

lt has been shown that the high-energy stretching vibrations tend to be bond-localized owing to the anharmonic couplings [27,41]. The rotational structure of the AsH stretching overtones exemplifies this transition from normal mode to local mode vibration. Fig. 4 shows the $J ^ { \prime } = 3$ levels of the $\left( n \ 0 \ 0 \right)$ states $( n = 1 , 2 , 3$ and 4). The $( 1 0 0 )$ state corresponds to the fundamental normal mode vibrational states $v _ { 1 } = 1 \left( \mathbf { A } _ { 1 } \right)$ and $v _ { 3 } = 1 ~ ( \mathrm { E ) }$ , which are separated by 11.26 cm−1 [29] in band center. The level structure shows obvious symmetric top characters and K is a ‘good’ quantum number. With the increase of vibrational quantum number, the lowest pair of $\mathbf { A } _ { 1 } / \mathbf { E }$ normal mode states (local mode pair) become more and more degenerate owing to the Darling–Dennison resonance. Consequently the rotational structure becomes complicated due to the strong rovibrational $r \mathrm { - } , l \mathrm { - }$ and a-resonances as demonstrated in the first stretching overtone. However, it is interesting that the rotational structure of the higher overtones exhibits simple patterns as shown in the (3 0 0) and (4 0 0) vibrational state. In these cases, the local mode pair become nearly degenerate, i.e. the separation of the band centers is $< 0 . 0 1 \ \mathrm { c m } ^ { - 1 }$ . The strong resonances due to the vibrationally off-diagonal $H _ { 2 2 }$ operators rearrange the rotational energy levels and lead to $\mathbf { A } _ { 1 ( 2 ) }$ –E degeneracy as shown in Fig. 4.
